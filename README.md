# Skills

A collection of skills I made for AI agents and models to speed up my workflow as well as yours.

This repo is a quick startup/setup guide for agentic IDE users. It points your agent to installable skill links and MCP config examples. Nothing special happens locally from reading the repo; your agent just learns where the skills live and can install the ones you choose.

This can be useful for all existing, as well as new users of any agentic IDE, but most especially Codex users.

## Quick Start

Paste this into your AI agent:

```text
$skill-installer install the listed skills at https://github.com/iice257/Skills
```

For a single skill:

```text
$skill-installer install https://github.com/iice257/Skills/tree/main/skills/frontend-skill
```

After installing skills in Codex, restart Codex so the new skills are picked up.

## What This Repo Is For

- Fast bootstrap for a new Codex setup.
- A public list of reusable skills with direct install links.
- A lightweight reference for MCP configs I commonly use.
- A shareable link for new users who want a useful starter kit without hunting through local files.

## Launch Workflow

Use `idea-launcher` and `x-publisher` together when a rough product idea needs to become a shareable repo:

1. Start with `idea-launcher` to interview the idea, cut the scope, generate the project plan, and create a runnable starter.
2. Build the smallest useful version and keep the README focused on the problem, demo, install path, and next step.
3. Use `x-publisher` to draft a short launch post or thread, validate character counts, and open a final composer link for manual review.

This keeps the repo useful before promotion and gives the launch post a concrete artifact to point at.
