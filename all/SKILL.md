# All Skills Index

Do not try to install this folder as one normal skill bundle.

The `all/` folder stores individual skill folders. To install everything, use:

`https://github.com/iice257/Skills/tree/main/categories/full-startup`

To install a specific skill, install that skill's direct folder under `all/`, for example:

`https://github.com/iice257/Skills/tree/main/all/idea-launcher`

If the user is unsure what they need, send them back to the root `SKILL.md` or `categories/SKILL.md` and ask what kind of setup they want.
