# Cursor Bundle

Use this when the user asks for Cursor provider skills from this repo.

This bundle currently has no installable `SKILL.md` paths in `paths.txt`.

1. Open `paths.txt` in this folder.
2. If it is empty, tell the user no installable Cursor `SKILL.md` files were found yet.
3. Do not invent Cursor skill paths.
4. If paths are added later, install each non-empty line at `https://github.com/iice257/Skills/tree/main/<path>`.
5. After any future install, tell the user: `Restart Codex to pick up the new skills.`
